# Backtest Review Tasks

## Review Backtest Report

Input: strategy description, simulation rules, data sources, costs, metrics, and report.

Output: findings, risks, missing tests, and production-readiness recommendation.

## Bias Checklist

Input: backtest configuration, data lineage, timestamps, labels, and universe construction.

Output: checklist covering lookahead, leakage, survivorship, restatements, parameter search, and benchmark selection.

## Robustness Test Plan

Input: current backtest result, known assumptions, and strategy constraints.

Output: test matrix for date windows, costs, slippage, universe filters, parameter perturbations, regime splits, and execution delays.

## Paper Trading Readiness Review

Input: backtest report, monitoring plan, risk limits, and implementation notes.

Output: readiness assessment with blockers, required controls, and handoff notes.
