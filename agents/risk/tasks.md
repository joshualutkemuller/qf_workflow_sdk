# Risk Agent Tasks

## Exposure Review

Input: holdings or weights and a factor model or return history.

Output: factor and exposure decomposition separating intended from unintended risk.

## Drawdown & Tail Profile

Input: return or equity history, or backtest output.

Output: drawdown depth/duration/recovery and left-tail characterization.

## Stress & Scenario Analysis

Input: strategy and a set of historical or hypothetical scenarios.

Output: behavior under each scenario with the exposures that drive it.

## Risk Limits & Monitoring

Input: strategy risk profile and mandate constraints.

Output: proposed risk limits as testable thresholds, monitoring metrics, and
breach actions.
