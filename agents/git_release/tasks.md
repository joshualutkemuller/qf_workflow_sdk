# Git & Release Agent Tasks

## Write Commit Messages

Input: a change set or diff.

Output: Conventional Commit message(s), scoped and honest about the change.

## Draft Pull Request

Input: the change and its owning spec.

Output: a PR description mapping requirements and acceptance criteria to what changed.

## Prepare Release

Input: the changes since the last release and the versioning policy.

Output: changelog entry and version bump consistent with the change.

## History Hygiene Review

Input: a branch or commit series.

Output: recommendations to make history small, coherent, and reviewable without
destroying shared work.
