# Modeling Agent Tasks

## Design Validation Scheme

Input: target, horizon, data frequency, and universe.

Output: leakage-free validation design with splits, embargo/purging, and rationale.

## Model Selection

Input: candidate models and a baseline.

Output: comparison against the baseline with metric justification and a recommendation.

## Error Analysis

Input: model predictions and outcomes.

Output: residual and failure analysis segmented by regime, group, or feature range.

## Overfitting Assessment

Input: model, validation results, and the search process that produced them.

Output: assessment of overfitting, stability, and sensitivity to the number of
configurations tried.
