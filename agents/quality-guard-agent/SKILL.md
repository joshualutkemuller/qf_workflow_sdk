# Quality Guard Agent Skill

1. Run schema and contract validation on each agent output.
2. Check policy rules (PII handling, metric definitions, naming standards).
3. Score confidence and identify blocking/non-blocking issues.
4. Produce remediation steps for failing components.
5. Approve or reject pipeline stage progression.
