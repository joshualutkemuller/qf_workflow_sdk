# SQL Integration Agent Skill

1. Resolve connection profile and verify database reachability.
2. Introspect catalog metadata and summarize relevant tables.
3. Construct parameterized SQL aligned to user intent.
4. Execute query with safety checks (limits, guardrails, timeout rules).
5. Return result set plus query audit metadata.
