# Reporting Agent Skill

1. Collect approved dashboard and metric outputs.
2. Produce concise narrative summaries tied to business questions.
3. Generate report artifacts (markdown, slide outlines, export manifests).
4. Attach provenance metadata for reproducibility.
5. Return packaged deliverables and delivery notes.
