# Tableau Agent Tasks

## Data Source Review

Input: a Tableau data source (connections, joins/blends, grain).

Output: a review of grain, join vs blend choices, and extract-vs-live suitability.

## Calculation Review

Input: LOD expressions and table calculations.

Output: verification against known totals, with corrected calculations.

## Visualization Review

Input: dashboard views.

Output: chart-choice and honest-encoding review, with fixes to misleading visuals.

## Publishing & Permissions Review

Input: a published workbook.

Output: a permissions and credential-handling review with safe fixes.
