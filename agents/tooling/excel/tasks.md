# Excel Agent Tasks

## Model Review

Input: a workbook and its intended purpose.

Output: findings ordered by severity covering structure, formula errors,
look-ahead, and hidden overrides, with fixes.

## Restructure Workbook

Input: a tangled model.

Output: an input/calc/output structure with named ranges, consistent formulas, and
reconciliation checks.

## Point-in-Time Audit

Input: a time-series spreadsheet.

Output: a look-ahead review of the layout and formulas, with corrections.

## Graduate-To-Code Assessment

Input: a high-risk or heavily-used workbook.

Output: an assessment of whether to rebuild the logic as versioned, tested code,
with the trade-offs and a migration sketch.
