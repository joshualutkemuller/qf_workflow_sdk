# Knowledge Curation Tasks

## Build Taxonomy

Input: an ingested corpus and its domains.

Output: a cross-domain taxonomy/ontology and a consistent tagging scheme.

## Deduplicate & Canonicalize

Input: overlapping documents on the same topics.

Output: near-duplicate clusters, a canonical item per topic, and links from the rest.

## Resolve Conflicts

Input: sources that contradict each other.

Output: the canonical source per conflict with the basis (authority, recency,
ownership) stated.

## Staleness & Gap Audit

Input: the curated base and consumer needs.

Output: staleness/supersedence flags and a list of coverage gaps.
