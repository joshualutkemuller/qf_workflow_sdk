# EDA Specialist Agent Skill

1. Validate dataset schema and row counts before analysis.
2. Produce numeric summary statistics (min/max/mean) for measurable fields.
3. Identify missingness patterns and unusual value ranges.
4. Generate concise insights tailored to the selected analytical mode.
5. Return EDA report artifacts and recommended next analytical actions.
